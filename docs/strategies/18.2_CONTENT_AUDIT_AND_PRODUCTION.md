# MS18 Phase 2 Strategy — Content Audit & Production

Status: Not started.
Parent milestone doc: [`docs/milestones/18_PRERELEASE_CONTENT_SEO.md`](../milestones/18_PRERELEASE_CONTENT_SEO.md)

**Goal:** Audit the existing Helpful Notes, make Adapt/Inspire/Skip decisions, and produce all planned content — moving directly from audit decision to writing for each piece.

## Content workflow

For every article produced in this phase:

1. **Author** — create `site/learn/<slug>.md` with front matter: `layout: article.njk`, `title`, `description`, `canonical: https://cryptozing.app/learn/<slug>/`
2. **Build** — `cd site && npx @11ty/eleventy`
3. **Stage** — verify at `http://<dev-server-LAN>/content/learn/<slug>/` via Sail
4. **Publish** — push to `main`; GitHub Actions builds and deploys to GitHub Pages automatically
5. **Verify** — confirm HTTP 200 at `https://cryptozing.app/learn/<slug>/`
6. **Sitemap** — add the URL to `site/sitemap.xml` with today's date as `lastmod`
