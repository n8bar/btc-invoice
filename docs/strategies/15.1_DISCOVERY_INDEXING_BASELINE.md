## 3. Establish ownership and submission in external tools
1. [x] [Shared] Choose the intended ownership model for `cryptozing.app` in Google Search Console and Bing Webmaster Tools.
   - Current decision on 2026-03-27: use a DNS-verified Google Search Console domain property for `cryptozing.app` so the root domain, future protocols, subdomains, and paths stay under one property as the placeholder eventually gives way to the live app. For Bing Webmaster Tools, prefer importing the verified Google property first for the initial baseline; only add separate Bing DNS verification later if Phase 1 or later rollout work proves we need Bing decoupled from the Google verification path.
2. [x] [Guided User] Verify `cryptozing.app` in Google Search Console.
   - Guided flow:
     - Open Google Search Console and choose `Add property`.
     - Select `Domain`, then enter `cryptozing.app` without `https://` or any path.
     - Use Google's DNS verification flow and copy the TXT record token it gives you. If Google offers the registrar-assisted auto-DNS flow for the current registrar, use it; otherwise add the TXT record manually at the DNS host for `cryptozing.app`.
     - Keep the TXT record exactly as shown, then return to Search Console and click `Verify`.
     - If verification does not succeed immediately, leave the TXT record in place and retry after propagation instead of changing the property type or switching to a URL-prefix workaround.
     - After success, leave the TXT record in DNS so the domain property stays verified.
   - Current result on 2026-03-27: Google Search Console domain-property verification succeeded for `cryptozing.app` using the DNS TXT record.
3. [x] [Guided User] Verify `cryptozing.app` in Bing Webmaster Tools.
   - Current result on 2026-03-27: Bing Webmaster Tools did not find the site through the Google import flow right away, so verification was completed directly with Bing's CNAME method instead. The authoritative and public DNS checks both showed `af754b184fb17694c7a30fc134e7c1c7.cryptozing.app CNAME verify.bing.com`, and Bing then accepted the site.
4. [x] [Guided User] Submit `https://cryptozing.app/sitemap.xml` in both tools.
   - Guided flow:
     - In Google Search Console, open the verified `cryptozing.app` domain property.
     - Open `Sitemaps`.
     - Submit `https://cryptozing.app/sitemap.xml`.
     - Treat Google as complete for this step once the sitemap is accepted or shows a normal in-progress state rather than a property/verification error.
     - In Bing Webmaster Tools, treat this step as complete once the same sitemap shows `Success` or an equivalent accepted-processing state.
   - Current result on 2026-03-27: Google Search Console shows the sitemap submission as `Success`, and Bing Webmaster Tools also shows `Success`.
